# Dino Game
***
Jogo desenvolvido em um bootcamp da digital inovation one.
Utilizando: html, css e javascript.

Jogo é inspirado no T-Rex Game, um jogo integrado ao navegador Google Chrome que pode ser acessado pressionando a barra de espaço quando estiver offline no Google Chrome.
